## ex05 for with if
* 실습파일 : ex05.py
* 이름이 `func05`인 함수를 다음과 같이 완성하시오
* 매개변수로 `a`와 `b`, `c` 를 받아 들이시오
* `a`, `b`는 정수로 `a` 보다 `b` 가 큰 값일 것임
* `c`는 `0` 또는 `1`
* 함수의 반환값:<br>
`c`가 `1` 이면 `a` 이상 `b` 미만인 홀수의 합<br>
`c`가 `0` 이면 `a` 이상 `b` 미만인 짝수의 합
* 제출하기 전 코드가 바르게 작동하는지 확인하시오
* 해당 파이썬 스크립트 파일에는 해당 함수만 제출
